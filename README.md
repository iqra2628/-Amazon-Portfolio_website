Amazon Clone
Overview
This project is an Amazon Clone, a frontend-only static website designed to replicate the look and feel of the Amazon homepage. It showcases the use of HTML and CSS to create a visually appealing and responsive layout.

Features
Homepage Layout

Header with logo, search bar, and navigation links
Hero section with promotional banners
Product categories and featured product sections
Footer with additional links and information
Product Listing Page

Grid layout for displaying products
Product cards with images, names, prices, and ratings
Hover effects for interactive product display
Product Detail Page

Detailed view of a single product
Product images with zoom effect
Description, pricing, and reviews section
Responsive Design

Mobile-friendly layout using media queries
Flexible grid and layout adjustments for different screen sizes
Technologies Used
HTML

Semantic HTML5 elements
Organized and well-structured markup
CSS

CSS3 for styling and layout
Flexbox and Grid for responsive design
Custom animations and transitions
